## Development

1. Install NodeJS 16.20.0.
1. Install yarn.
   ```
   npm i -g yarn
   ```
1. Install node modules.
   ```
   yarn
   ```
1. Edit you GoogleAppScriptId in file `.clasp.json`.
1. Login.
   ```
   yarn login
   ```
1. Start watch.
   ```
   yarn watch
   ```
1. Run function on site:
   `https://script.google.com/u/1/home/projects/ <GoogleAppScriptId> /edit`.
